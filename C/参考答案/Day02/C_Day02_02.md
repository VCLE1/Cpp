### 拓展

C/C++ 中有一个函数 `atoi`，它可以将一个字符串转换为 32-bit 的有符号整数，请实现 `myAtoi` 函数完成同样的功能。

1. 忽略前面的空白字符。
2. 检查下一个字符是否是 '+' 和 '-'，如果没有出现，默认为正数。
3. 读取下一个字符，直到遇到非数字字符，或者到字符串的末尾。
4. 将读入的数字字符转换为对应的整数 (i.e. "123" -> 123, "0023" -> 32). 如果没有读取任何一个数字，则结果为 0.
5. 如果超出了整数的表示范围 [-2^31^, 2^31^ - 1], 则将小于 -2^31^ 的值 转换为 -2^31^，大于 2^31^ - 1 的值转换为 2^31^ - 1.

```c
#include <ctype.h>
#include <limits.h>

int myAtoi(char* s){
    // skip blank characters
    while (isblank(*s))
        s++;
    int sign = 1;
    if (*s == '-') {
        sign = -1;
        s++;
    } else if (*s == '+') {
        s++;
    }
    
    long long result = 0;
    while (isdigit(*s)) {
        result *= 10;
        result += *s - '0';
        s++;
        if (result > INT_MAX)
            break;
    }
    
    result = sign * result;
    if (result > INT_MAX)
        return INT_MAX;
    else if (result < INT_MIN)
        return INT_MIN;
    else return result;
}
```
