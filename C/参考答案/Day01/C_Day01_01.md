### 拓展

下面是 Hello World 程序对应的汇编代码 (Intel 格式)，请阅读下面的汇编代码，看是否能找到和 C 语言代码的对应关系。

```c
// helloworld.c
#include <stdio.h>

int main(void) {
    printf("Hello world.\n");
    return 0;
}
```

```assembly
# assembly
.LC0:
        .string "Hello world."
main:
        push    rbp
        mov     rbp, rsp
        mov     edi, OFFSET FLAT:.LC0
        call    puts
        mov     eax, 0
        pop     rbp
        ret
```
