### 题目

(a) 合并两个有序链表。要求合并后的链表也是有序的 (要求: 不能额外申请堆内存空间)。

```c
typedef struct node_s {
	int val;
    struct node_s* next;
} Node;

Node* mergeTwoLists(Node* list1, Node* list2);
```

```c
Node* mergeTwoLists(Node* list1, Node* list2){
    Node node = {0, NULL}; // dummy node
    Node* head = &node;
    Node* curr = head;
    
    while (list1 && list2) {
        if (list1->val < list2->val) {
            curr->next = list1;
            curr = curr->next;
            list1 = list1->next;
        } else {
            curr->next = list2;
            curr = curr->next;
            list2 = list2->next;
        }
    }
    
    if (list1) curr->next = list1;
    if (list2) curr->next = list2;
    
    return head->next;
}
```
