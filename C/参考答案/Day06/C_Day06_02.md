(b) 利用 getchar() 实现类似 gets_s 的功能。

```c
char *read_line(char *str, size_t n);
```

```c
char *read_line(char *str, size_t n) {
    int i = 0, ch;
    while ((ch = getchar()) != '\n' && i < n) {
        str[i++] = ch;
    }
    str[i] = '\0';
    
    return str;
}
```
